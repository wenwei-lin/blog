The python version, packages or modules you use in each python project will often differ from others. Some projects may even require a specific version of package to escape bugs. If you install all the packages globally, you are likely to encounter package conflict someday.

***You need python virtual environment!***

## What is python virtual environment?

Python virtual environment is a cooperatively isolated runtime environment that allows Python users and applications to install and upgrade Python distribution packages without interfering with the behaviour of other Python applications running on the same system.
